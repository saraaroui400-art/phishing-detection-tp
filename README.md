# 🎣 Phishing Detection TP

## 📌 Description
Ce projet est un TP de Cybercriminalité qui simule une attaque de phishing et développe des techniques de détection.

## 🎯 Objectif
- Comprendre le fonctionnement du phishing
- Analyser les headers SMTP
- Détecter les emails malveillants avec YARA / SIEM

## 🛠️ Outils utilisés
- HTML
- CSS
- JavaScript
- Wireshark
- SET Toolkit
- YARA
